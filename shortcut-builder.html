Part A: Setup

Create shortcut and format the date

Create New Shortcut

Add: Current Date

3
Add: Format Date
Add Action → search "Format Date" → tap it
• Input: Current Date
• Date Format: tap → Custom → type exactly: yyyy-MM-dd
Format
Format [Current Date] → Custom: yyyy-MM-dd
This converts "March 23, 2026" to "2026-03-23"
4
Add: Text (build URL)
Add Action → search "Text" → tap the plain Text action

Important: Tap the "Copy URL" button above, then paste into the Text field. Then:
1. Position cursor at the END (after date=)
2. Tap the variable button above keyboard
3. Select Formatted Date
Text
https://script...?action=getHealthExport&key=...&date=Formatted Date
💡 The blue pill "Formatted Date" should appear right after &date= with no spaces
5
Add: Get Contents of URL
Add Action → search "Get Contents of URL" → tap it

• Expand (tap arrow) → Method: GET
Network
Get contents of [Text]

Variable

Part B: Log Health Data

Repeat this pattern 7 times — one for each health type

🔁 The Pattern (repeat for each type below)
Each health metric follows the same 3-action pattern:
Step A
Get Dictionary Value → key: [KEY] in [HealthData]
Step B
If [Dictionary Value] is greater than 0
Step C (inside If)
Log Health Sample → Type: [TYPE] → Value: [Dictionary Value] → Unit: [UNIT]
💡 Each "If" automatically adds "Otherwise" and "End If" — leave those empty
1️⃣ Calories
Get Dictionary Value: key = dietaryCalories in HealthData
If: greater than 0
Log Health Sample: Type = Dietary Energy · Unit = kcal · Value = Dictionary Value · Date = Current Date
2️⃣ Protein
Get Dictionary Value: key = dietaryProtein in HealthData
If: greater than 0
Log Health Sample: Type = Protein · Unit = g · Value = Dictionary Value · Date = Current Date
3️⃣ Carbs
Get Dictionary Value: key = dietaryCarbs in HealthData
If: greater than 0
Log Health Sample: Type = Carbohydrates · Unit = g · Value = Dictionary Value · Date = Current Date
4️⃣ Fat
Get Dictionary Value: key = dietaryFat in HealthData
If: greater than 0
Log Health Sample: Type = Total Fat · Unit = g · Value = Dictionary Value · Date = Current Date
5️⃣ Active Energy (Workout)
Get Dictionary Value: key = activeEnergy in HealthData
If: greater than 0
Log Health Sample: Type = Active Energy · Unit = kcal · Value = Dictionary Value · Date = Current Date
6️⃣ Body Weight
Get Dictionary Value: key = bodyMass in HealthData
If: greater than 0
Log Health Sample: Type = Body Mass · Unit = kg · Value = Dictionary Value · Date = Current Date
7️⃣ Water
Get Dictionary Value: key = waterMl in HealthData
If: greater than 0
Log Health Sample: Type = Water · Unit = mL · Value = Dictionary Value · Date = Current Date

Part C: Finish

Notification + automation

8
Add: Show Notification
Add Action → search "Show Notification"
• Title: Health Sync Complete
• Body: Diet data synced to Apple Health
9
▶ Test It!
Tap the Play ▶ button at the bottom of Shortcuts.

First time: iOS asks for Health permissions — tap Allow for all types.

Then check: Health app → Browse → Nutrition → you should see today's data!
⚠️ If you see "There was a problem" — go back to Step 4 and check your URL has no spaces or line breaks
10
🤖 Automate (Run Every Night)
1. Go to Automation tab in Shortcuts
2. Tap + → Time of Day
3. Set: 9:00 PM · Repeat: Daily
4. Tap Next
5. Tap Run Shortcut → select "Sync Diet to Health"
6. Turn OFF "Ask Before Running"
7. Tap Done
💡 Now it runs silently every night. You never touch it again.
🎉
All Done!
Your diet data will sync to Apple Health every night at 9 PM
